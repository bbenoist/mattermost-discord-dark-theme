# Mattermost "Discord" dark theme
A theme for Mattermost trying to approach Discord's dark theme.

## Usage
In Mattermost:
1. click on User menu
1. select Account Settings
1. select Display
1. edit Theme
1. copy and paste the string below in the appropriate box
1. hit save
1. enjoy
